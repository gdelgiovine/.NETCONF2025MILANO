# .NETCONF2025MILANO
.NET Conference Milano 2025 Samples 

Contenuto:
Presentazione NetConference2025.pdf
Presentazione NetConference2025.pptx
Wisej3BackgroundTasks.zip  - Gestione delle operazioni in background in Wisej.NET
Wisej3HotelDemo.zip - Demo gestione prenotazioni camere Hotel con dashboard prenotazioni
WisejCSSHTMLDemo.zip - Uso di CSS ed HTML in Wisej.NET
WisejHybridPhoto.zip - Uso della fotocamera nelle applicazioni Wisej.NET Hybrid
WisejResponsiveProfile.zip - I tools per la creazione di UI responsive in Wisej.NET

ed effettuare il restore del database SQLServer HotelDemo. Il database HotelDemo è in versione SQLServer 2022.

Per sperimentare o crearvi la vostra implementazione del framework MVVM PasseroFramework scaricate la relativa soluzione 
dal suo repository https://github.com/gdelgiovine/PasseroFramework
